# Fisher's Theorem in Multiple Linear Regression

This repository contains a theoretical study of Fisher's theorem in the
context of multiple linear regression models.

The document presents a derivation of the sampling distributions of the
estimators of the regression coefficients under the assumption of
normally distributed errors.
